### 리액트 컴포넌트에서 undefiend만 반환하여 랜더링하는 상황을 만들지 않아야 한다.

### 예를 들어 다음과 같은 상황은 에러를 발생시킨다.

```javascript
import React from 'react;

function App() {
    const username = undefined;
    return username
}

export default App;
```
